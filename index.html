QUICK START // SITE GUIDE

How To Use This Site

Use this page as the quick guide before opening the dossier or referee console. The goal is to make setup, upload, and live table use easy to understand at a glance.

Start Here

Choose Your Entry Point

Use Load File for an existing dossier, Create New Character for a blank sheet, or Referee Access for the GM console. The launcher feeds into the same shared toolkit after the transition.

Files

What You Can Import

The site supports character .txt, item .txt, and bundled dossier .zip files. The format guide above is the template to follow when preparing files for upload.

Room Link

Connecting Players And GM

Open the GM page, enter a room ID, then connect player dossiers to that same room from the dossier page. This lets the referee monitor linked characters and use the live table tools.

NPC / Mooks

Use External Chatbot AI

For fast random NPCs, mooks, or one-off enemies, the intended workflow is to use an outside chatbot AI, then format the result to match the upload template shown above and import it as a dossier file.

Session Flow

Typical Use

Load or create characters, connect the room if you want live GM support, then use the dossier for player-side tracking and the GM page for monitoring, NPC handling, and combat-side control.

Build Note

Development

This project was built with roughly 80% AI assistance during development, but for actual use the important part is the workflow: upload correctly formatted files, link the room when needed, and use external chatbot generation for random content.

After you upload or confirm a new character, the system runs a short "CHIPPIN IN" glitch load before opening the dossier.
