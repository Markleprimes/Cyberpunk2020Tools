PROJECT README // PUBLIC NOTE

What This Is

This page doubles as the public quick-read for the project, so someone landing here can understand the tool before opening the dossier or referee console.

Project

Interface Suite For Table Play

This project is a static browser toolkit for Cyberpunk 2020 sessions. It combines a player dossier, a referee console, animated roll systems, inventory tracking, room sync, and a Netrunner minigame into one themed interface.

Disclosure

AI-Assisted Build

About 80% of this project was built with AI assistance. That includes a lot of scaffolding, implementation, and iteration work. The direction, feature choices, testing, and editing decisions were still human-led.

Use Case

Built For Actual Sessions

The goal is not "perfect software purity." The goal is a stylish, fast, useful table tool that helps with character handling, combat pacing, GM monitoring, and hacking scenes.

NPC / Mooks

Use External Chatbot AI

For fast random NPCs, mooks, or one-off enemies, the intended workflow is to use an outside chatbot AI, then format the result to match the upload template shown above and import it as a dossier file.

Pages

Three Main Entry Points

Launcher for access and upload, Dossier for player-side tracking, and Referee Access for live room monitoring and combat-side control.

Publishing Note

Transparent On Purpose

This is being published with the AI usage stated openly. I would rather be honest about how it was made than hide the process behind fake "handmade" presentation.
